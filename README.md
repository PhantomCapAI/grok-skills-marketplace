# SkillForge ⚡

**Open Source Marketplace for Grok & Claude Skills**

Browse, discover, and install high-quality skills for Grok (xAI), Claude, and other AI systems.

Built by **Phantom Capital** (@PhantomCap_ai)

## ✨ Features
- Clean dark UI with Tailwind CSS (via CDN)
- Live search + category filtering
- Skill preview modal with one-click copy install instructions
- Designed for Grok's `~/.grok/skills/` structure
- Fully static — works offline after download

## Quick Start
1. Clone or download the repo
2. Open `index.html` in any browser
3. Start browsing & installing skills

## Grok Skills Folder Structure
```bash
~/.grok/skills/
├── phantom-finance-analyzer/
│   ├── skill.md
│   └── instructions.md
├── research-beast/
└── ...
```
